# TF Tasks for Week 5

The core concept students needed to understand

Sentimental analysis may be new to some students. The core concept include used rule-based and ML approaches to do sentimental analysis.

Where students are most likely to struggle

Rule-based is not as straightforward as it seems. How we assign the score and classify will affect the results a lot. It's hard to decide in short time.


Where AI was helpful vs misleading

AI is very helpful in understanding existing code and come up with test cases. But human review is very important to validate the tricky cases when sarcasm is involved.


One way they would guide a student without giving the answer

Identify one failed prediction and trace through the processing logic. Also it would be helpful to explain how the logistic regression works so students are not confused with the high accuracy if they train all data without splitting test data and train data.
